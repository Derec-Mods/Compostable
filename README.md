## Compostable

_This was created for use on my forever world, called "derex smp". Check it out in action, server ip: `mc.longhorns.dev`_

Rotten flesh and other things should be compostable, this is a lightweight plugin to allow them to be done so, as configurable as can be. Most importantly, automation is possible via hoppers, so you can make auto-bone meal farms with zombie spawners!

Designed to be simple, server-friendly, and as vanilla as possible.
